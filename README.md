# local-IP-sciprt

**How to install**

    sudo wget https://github.com/CheemsBread505/local-IP-sciprt/releases/download/v1.0.0/localIP && mkdir /opt/localIP/

**And then:**

    sudo cp localIP /opt/localIP/ && rm localIP

**To test it out use:**

    localIP
**It should print out your local IP adress.**
